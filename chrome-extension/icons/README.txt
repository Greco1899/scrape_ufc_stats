UFC Predictor Data Collector - Icons

You need to create PNG icons in the following sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Suggested design:
- Use the letters "UFC" or a simple octagon shape
- Primary color: #1a73e8 (blue)
- Background: transparent or white

You can create these using any image editor or online tool like:
- https://www.canva.com
- https://www.favicon.io
- GIMP, Photoshop, etc.

For quick testing, you can use simple colored squares as placeholders.
